# ⚡ Goal-Force

A modern, full-stack React-based goal and streak tracking application built with a premium UI.

## 🚀 Features
* **Dynamic Dashboard:** Tracks daily streaks and overall goal progression.
* **Smart Categorization:** Supports micro-goals (10 days) up to lifetime achievements.
* **Premium UI/UX:** Features a sleek dark/light mode toggle with smooth keyframe animations.
* **Local Storage Integration:** Persists user sessions and goal data securely.

## 🛠️ Tech Stack
* **Frontend:** React, HTML5, CSS3
* **Data Visualization:** Recharts
* **Architecture:** Modular component design with separated logic (`utils.js`)

## 💡 Why I Built This
I developed Goal-Force to demonstrate clean code architecture, state management in React, and my ability to design production-ready, highly aesthetic user interfaces from scratch.
